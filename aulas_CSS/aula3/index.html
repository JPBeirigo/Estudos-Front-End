Agrupamento de Seletors

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula (,)

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco deCSS e reutilizar as mesmas regras!
